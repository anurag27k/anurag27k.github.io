User Flows

There are several explored flows in this application. Some of them are discussed below.
The image quality has been compromised due to creation of gifs. Clear images can be accessed Here


#1 Create a roadmap

The tool helps people to create a roadmap with presets instead of making it from scratch. The template allows the user to add the number of tasks as well as the type of tasks. People can create items and sections. There are also bubbles which prompt people to fill details. Details of a task can be found by clicking on an item.






#2 Creating Milestones

Milestones or checkpoints could be added on the Gantt chart by clicking on the black line at the top. Name of the milestone and recurrence can be added too. The milestone appears as black line in the Gantt chart. The milestone and be clicked and dragged to move it to another date. All these can be seen at the Milestones page.






#3 Future Notes and Goals

In the left menu we can find this option. This includes all the goals and projects which might be taken in the future but their timeline is unknown. They might be taken up next year, or the year after that or never. The goals be sent directly to the timeline in case we wish to plan for that in the current timeline.






#4 Bucket view

This is a gamefied way of adding values to different tasks. The buckets can be created on the based on Quarters, Teams, Impact, Priority and Size. People can drag from the current items list to the buckets having different values. Below we can see buckets based on Priority: Low, Medium, High.






#5 Finalize Roadmap

The user can see the finalized roadmap in different views. We have the summary view where we see the list of projects in a list form with all its details. We have a Quarters view where we can see the items taken up in each quarter with section segregation. We also have a FTE view(Full time Employee) view where we can see how every employee is occupied in each quarter.






#6 Employee Management

In the employee management page we can understand details about an employee. We can note down the skill set an employee currently has and the skill set the employee wishes to learn. A section for analytics has also been included. Since there are a bunch of auxiliary tasks apart from the main tasks, a section for the same is created where a person can add their auxiliary tasks.
