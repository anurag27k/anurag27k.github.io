SKYL.AI

People10 Project

This website automates End-to-End machine learning workflow. There are 3 parts in here namely collaborator app, marketing website and branding.

01

Problem statement

Allow users to complete variety of tasks like categorizing, labelling and collecting data. Create a mobile application which is similar to the existing web application.


02

User profiles

This application has two types of user profiles.
  • Job creator
  • The person assigned to do the job


03

Solution features

An application which allows to:

See the new, ongoing and complete jobs in a screen
Complete the type of job with ease of mobile interface
Able to recieve notifications about new jobs
Receive notifications about messages from creator

Able to see the progress of the job
Able to skip (if possible) the job or part of the job
Able to find instructions and support for the application
